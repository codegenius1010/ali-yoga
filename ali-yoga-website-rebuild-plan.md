# Ali Yoga Website Rebuild Plan
## Modern SEO-Driven Rebrand + Next.js Build Brief

## 1. Project Objective

Build a modern, SEO-driven replacement website for Ali Yoga / Hydrate Ali Yoga that feels elevated, warm, holistic, premium, and conversion-focused.

The new site should position Ali as a yoga teacher, private coach, retreat host, and specialist in yoga for golfers. The site should generate leads for:

1. Private 1:1 yoga coaching
2. Yoga for golfers
3. Online yoga courses
4. Retreats
5. Local classes / workshops
6. Email list signups
7. Booking inquiries

The website should feel more like a premium wellness brand than a simple yoga portfolio.

---

## 2. Recommended Brand Direction

### Brand Positioning

Ali Yoga should become a modern wellness brand centered around:

**“Yoga for freedom, presence, and performance.”**

The brand should balance three pillars:

1. **Holistic yoga** — asana, breathwork, meditation, spiritual discipline
2. **Performance yoga** — flexibility, mobility, balance, focus, especially for golfers
3. **Transformational experience** — retreats, personal coaching, embodiment, inner freedom

### Suggested Brand Name Options

Keep one of these:

1. **Ali Yoga** — simplest, personal, clear
2. **Ali Yoga Method** — stronger for courses and private coaching
3. **Hydrate Ali Yoga** — keep only if “Hydrate” is strategically meaningful
4. **Ali Yoga Wellness** — broader wellness positioning
5. **Ali Yoga Retreats & Coaching** — clearer but less elegant

Recommendation: use **Ali Yoga Method** as the brand architecture, with **Ali Yoga** as the public-facing logo text.

Example:
- Logo: Ali Yoga
- Tagline: The Yoga Method for Freedom, Focus & Flow

### Typography

Use elegant, modern fonts:

- Headings: `Cormorant Garamond`, `Playfair Display`, or `Libre Baskerville`
- Body: `Inter`, `Lato`, or `DM Sans`
- Accent script, optional and minimal: `Allura` or `Cormorant Italic`

Recommended:
- Heading: `Cormorant Garamond`
- Body: `Inter`

### Logo Direction

Create a clean text-based logo with a subtle symbol.

Possible logo concepts:

1. **Ali Yoga** wordmark with a small sun rising through a lotus
2. **Ali Yoga** wordmark with a flowing “A” shaped like a yoga posture
3. **Ali Yoga Method** with a minimal circular breath/sun mark
4. A small line-art lotus + sun + wave symbol

Logo should work in:
- Horizontal header
- Square social icon
- Favicon
- Footer mark
- Light and dark backgrounds

## 6. SEO Strategy

### Primary SEO Themes

1. Yoga for golfers
2. Private yoga coaching
3. Online yoga courses
4. Yoga retreats
5. Breathwork and meditation
6. Holistic yoga teacher
7. Local yoga instruction if geography is relevant

### Page-Level SEO Requirements

Each page must include:

- Unique title tag
- Unique meta description
- H1 with primary keyword
- Clear H2/H3 structure
- Internal links
- Optimized image alt text
- Open Graph title/description/image
- Canonical URL
- JSON-LD schema
- Fast image loading
- Mobile-first design

### Recommended Schema

Use JSON-LD for:

- Organization
- Person
- LocalBusiness or YogaStudio if there is a real service location
- Service pages
- Course pages
- FAQPage
- BreadcrumbList
- BlogPosting
- Event for retreats or workshops

### Technical SEO

Include:

- `/app/sitemap.ts`
- `/app/robots.ts`
- metadata objects or `generateMetadata`
- canonical URLs
- semantic HTML
- accessible buttons and links
- optimized images through Next/Image
- clean URLs
- 301 redirect map from old URLs to new URLs

Redirects:
- `/about-me` → `/about`
- `/classes` → `/yoga-for-golfers`
- `/retreats` → `/retreats`
- `/testimonials` → `/testimonials`
- `/member-site-homepage-1` → `/online-yoga-courses`
